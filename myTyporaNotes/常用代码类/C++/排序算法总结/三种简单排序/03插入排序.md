# 算法思想

- 对基本有序的数组是最好的，稳定

- 好像打牌抓牌时一样，抓到比前面数小的，就往前放
- 跟冒泡排序差不多，只不过冒泡排序是直接冒到头，而插入排序，当遇到前一个比这位小的时候就不会冒了

```c++
#include <iostream>
using namespace std;

int main() {
	int arr[] = { 3,5,1,25,6,3,9,0,-1 };
	int len = sizeof(arr) / sizeof(int);
	
	for (int i = 1; i < len; i++)
	{
		for (int j = i; j > 0 && arr[j] < arr[j-1]; j--)
		{
				int tem = arr[j-1];
				arr[j-1] = arr[j];
				arr[j] = tem;
		}
	}

	for (int k = 0; k < len; k++) {
		cout << arr[k] << " ";
	}
	return 0;
}
```
